# **App Name**: ScholarSphere

## Core Features:

- Dashboard: Display an overview of the user's academic life including attendance, upcoming classes, nearest exam, and pending assignments.
- Timetable Management: Users can add, edit, and manage their class timetable, including subjects, teachers, timings, and classroom details, all saved to Firestore under their UID.
- Attendance Tracking: Automatic attendance tracking integrated with the timetable, allowing users to mark attendance and view attendance percentages.
- Assignment Manager: Add and manage assignments with titles, subjects, descriptions, and due dates, sorted and displayed for easy access, with reminders and optional local notifications.
- Exam Scheduler: Schedule exams with subject, date, time, venue, and type, with a countdown to the next exam displayed prominently.

## Style Guidelines:

- Primary color: Dark purple (#301934) for an intellectual and sophisticated feel in dark mode.
- Background color: Dark grey (#121212) for a modern dark mode aesthetic.
- Accent color: Light purple (#A020F0) to provide contrast and highlight key elements in dark mode.
- Headline font: 'Space Grotesk', sans-serif, for a modern and slightly technical feel, and 'Inter' for body.
- Use clean, line-based icons with a neon or glowing effect for a modern and consistent look in dark mode.
- Employ a modern UI with soft gradients and rounded cards for a smooth, engaging experience, optimized for dark mode with appropriate contrast.
- Implement subtle transitions and animations for user interactions to enhance usability, ensuring they are visible and effective in dark mode.
